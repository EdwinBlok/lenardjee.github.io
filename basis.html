De basis

De basis van deze website is gecreëerd door GitHub. Github is een site waar je makkelijk bestanden kunt sorteren en combineren door een repository aan te maken. Een repository is een project waarin je bestanden kunt aanmaken, bewerken en uploaden. Ook is het mogelijk om hele repositorys te kopiëren.

As je een repository hebt gemaakt is het tijd om bestanden aan te maken die de site bouwen. Het is hierbij van belang dat je bestanden de goede naam geeft. Er zijn verschillende soorten bestanden die gebruikt kunnen worden om de site te bouwen, namelijk .css bestanden, .js bestanden en .html bestanden.
