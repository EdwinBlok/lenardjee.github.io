Welkom!

Deze site hebben wij gemaakt om jullie iets bij te brengen van hoe je een site maakt met behulp van GitHub. GitHub is een site waarbij je projecten aanmaken waarin je bestanden kunt maken. Voor deze site hebben wij 2 verschillende projecten aangemaakt, 1 project om de broncode in te zetten en 1 project om de site zelf mee te maken. In GitHub kun je heel makkelijk een site maken door een project aan te maken met als naam eerst de gebruikersnaam van het account en daar achter .github.io wij hebben dus als projectnaam van het project dat de site maakt lenardjee.github.io
